# EVRT Research Framework v3.1 Release Notes

EVRT Research Framework v3.1 formalizes the repository as a structured phenomenological metrology and reproducibility framework.

## Highlights

- Updated framework-centered README with visual cover presentation
- Added modular experimental package infrastructure
- Added roadmap and version documentation
- Expanded diagnostics and scaling infrastructure
- Strengthened conservative scientific-position language
- Added null-test-first reproducibility workflows

## Scientific Position

This release does not claim experimentally verified anomalous-force behavior. It supports falsifiability-oriented investigation, uncertainty-aware diagnostics, and reproducibility-focused metrology.
