# Metrology and Experimental Protocol Papers

This folder contains precision experimental methodology papers associated with the EVRT research framework.

These works focus on:

- residual-force metrology
- artifact discrimination
- uncertainty-aware diagnostics
- torsion-balance methodology
- reproducible experimental architecture
- null-hypothesis stress testing

The metrology layer represents the transition from conceptual phenomenology toward experimentally testable precision measurement workflows.

---

# Scientific Position

The EVRT metrology framework does not claim experimentally verified anomalous propulsion or confirmed vacuum-engineering effects.

Instead, these works establish structured methodologies for:

- high-sensitivity testing
- artifact rejection
- uncertainty quantification
- reproducibility analysis
- conservative residual-force interpretation

Null results are considered scientifically valuable because they constrain possible EVRT-like response amplitudes.
