# Simulation and Numerical Modeling Papers

This folder contains numerical modeling, simulation-guided investigation, and finite-element analysis papers associated with the EVRT research program.

These works explore:

- electromagnetic field asymmetry
- stress redistribution
- resonant cavity behavior
- parameter-space trends
- scaling relationships
- simulation-guided experimental planning

The simulation layer provides exploratory numerical support for EVRT-compatible phenomenological investigation.

---

# Scientific Position

The simulation studies contained in this folder are illustrative and phenomenological unless explicitly tied to experimentally calibrated systems.

These works do not establish experimentally verified anomalous-force effects.

Instead, they are intended to support:

- sensitivity estimation
- parameter-space exploration
- scaling-law visualization
- geometry comparison
- uncertainty-aware interpretation
- experimental protocol development
