# Overview and Framework Synthesis Papers

This folder contains broad synthesis and framework-integration papers associated with the EVRT research program.

These works integrate multiple EVRT research layers into unified phenomenological and experimental architectures.

The overview layer connects:

- foundational theory
- simulation studies
- constraint analysis
- experimental methodology
- scaling-law development
- residual-force diagnostics

---

# Scientific Position

These works emphasize:

- falsifiability
- conservation-law consistency
- uncertainty-aware interpretation
- simulation-guided metrology
- artifact discrimination
- reproducibility

The overview papers do not claim experimentally verified anomalous-force phenomena or validated reactionless propulsion systems.

Instead, they provide structured frameworks for disciplined investigation of nonequilibrium electromagnetic systems.
