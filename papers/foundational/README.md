# Foundational Papers

This folder contains foundational and conceptual papers associated with the EVRT (Emergent Vacuum Response Theory) research program and the broader Coherent Nonequilibrium Electromagnetic Systems framework.

These works establish the phenomenological basis for investigating whether structured nonequilibrium electromagnetic systems may exhibit measurable residual stress-energy effects under precision experimental conditions.

The foundational layer emphasizes:

- emergent vacuum-response hypotheses
- nonequilibrium electrodynamics
- phenomenological field asymmetry
- coherent electromagnetic systems
- falsifiability-oriented framework development
- conservative theoretical interpretation

---

# Scientific Position

The foundational EVRT papers do **not** claim:

- experimentally verified anomalous forces
- validated antigravity phenomena
- reactionless propulsion
- confirmed vacuum-engineering effects
- experimentally established new physics

Instead, these works develop constrained phenomenological frameworks intended to guide:

- simulation studies
- parameter-bound analysis
- precision experimental protocols
- residual-force metrology
- artifact discrimination workflows
