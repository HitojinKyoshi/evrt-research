# Repository Structure

## Overview

This repository is organized as a structured research framework for exploratory investigations into coherent nonequilibrium electromagnetic systems and precision residual-force metrology.

---

## Directory Layout

### papers/

Formal manuscripts, publication assets, and source packages.

Includes:
- foundational theory papers,
- metrology frameworks,
- constraint-analysis studies,
- scaling-law papers,
- and overview publications.

---

### analysis/

Analysis methodologies related to:
- uncertainty budgeting,
- artifact discrimination,
- Bayesian classification,
- and residual-force interpretation.

---

### experiment/

Experimental workflows and metrology architectures.

Includes:
- torsion-balance concepts,
- geometry reversal protocols,
- calibration notes,
- vacuum testing procedures,
- and reproducibility templates.

---

### simulations/

Exploratory computational models and toy simulations.

Includes:
- scaling-law demonstrations,
- Monte Carlo examples,
- parameter sweeps,
- and stress-analysis tools.

---

### figures/

Research graphics and workflow diagrams.

Includes:
- scaling plots,
- diagnostic diagrams,
- roadmap figures,
- and metrology workflows.

---

### docs/

Framework documentation and explanatory materials.

Includes:
- research philosophy,
- EVRT overview documents,
- repository guidance,
- and roadmap notes.

---

### data_templates/

Templates intended for structured experimental logging and reproducible data collection workflows.

---

## Scientific Position

The repository is intended to support:
- falsifiable phenomenological research,
- precision metrology,
- uncertainty-aware interpretation,
- and systematic artifact rejection.

No verified anomalous-force generation or new physical effect is presently claimed.
