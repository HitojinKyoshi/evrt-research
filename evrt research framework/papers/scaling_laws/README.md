# Scaling-Law and Parameter-Space Papers

This folder contains EVRT papers focused on residual-force scaling, parameter-space analysis, sensitivity estimation, and canonical normalization frameworks.

These works investigate how candidate residual signatures may scale with:

- resonator Q-factor
- field strength
- asymmetry geometry
- power input
- energy density
- experimental sensitivity limits

---

# Scientific Position

The scaling-law framework is phenomenological and exploratory.

These papers do not establish experimentally verified anomalous-force effects.

Instead, they develop structured methodologies for:

- scaling comparison
- parameter normalization
- upper-bound interpretation
- sensitivity planning
- uncertainty-aware residual analysis

---

# Representative Topics

Typical topics include:

- force-versus-Q scaling
- signal-to-noise analysis
- effective susceptibility normalization
- parameter-space mapping
- residual-force upper bounds
- scaling-law consistency
- sensitivity-floor estimation
