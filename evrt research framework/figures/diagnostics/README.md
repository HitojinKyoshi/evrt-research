# Diagnostic Figures

This folder contains diagnostic visualization assets associated with the EVRT (Emergent Vacuum Response Theory) research framework.

These figures support:

- artifact discrimination
- uncertainty-aware metrology
- residual-force inference
- noise-floor evaluation
- Bayesian-style diagnostic classification
- sensitivity-bound interpretation

The figures in this directory are intended to help distinguish potential residual signatures from conventional experimental artifacts.

- uncertainty-budget charts
- noise-floor comparisons
- force-compass diagnostics
- environmental sensitivity plots
- artifact-classification graphics
