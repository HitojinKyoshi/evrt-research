# Experimental Figures

This folder contains publication-style experimental schematics and apparatus visualization assets associated with the EVRT research program.

These figures are intended to communicate:

- apparatus geometry
- torsion-balance layouts
- calibration workflows
- experimental configurations
- environmental isolation concepts
- residual-force metrology architectures

---

# Included Figure Types

Typical figures include:

- torsion-balance diagrams
- apparatus overview graphics
- calibration schematics
- resonator-layout illustrations
- experimental workflow visuals

# Scientific Position

These figures are conceptual and illustrative unless explicitly identified as experimentally calibrated engineering designs.

The EVRT framework does not claim experimentally verified anomalous propulsion or confirmed vacuum-engineering effects.

Instead, the figures support structured experimental methodology, artifact-aware metrology, and reproducible testing workflows.
