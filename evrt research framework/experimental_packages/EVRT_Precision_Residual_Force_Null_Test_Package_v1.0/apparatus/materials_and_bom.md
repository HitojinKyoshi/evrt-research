# Materials and Bill of Materials

## Core Components

- Acrylic base platform
- Nonconductive fishing-line suspension
- Lightweight torsion arm
- Aluminum or copper test plates
- Adjustable electrode spacing mounts
- Laser pointer
- Projection wall/screen
- Variable DC power supply
- Lightweight wiring (30 AWG preferred)
- Nonconductive mounting hardware

## Optional Upgrades

- Vacuum enclosure
- Environmental isolation enclosure
- Temperature monitoring
- Precision current sensing
- Optical displacement camera tracking
