# EVRT Precision Residual-Force Null-Test

## Pre-Run Experimental Checklist

---

# 1. Mechanical Setup Verification

* [ ] Torsion arm securely mounted and balanced
* [ ] Suspension line free of twists or tangles
* [ ] Arm oscillation damping acceptable
* [ ] No visible contact between moving components and surroundings
* [ ] Base platform leveled and mechanically stable
* [ ] Electrode spacing verified (2 cm / 5 cm configuration)
* [ ] Mass distribution symmetric except intended test asymmetry
* [ ] Wiring strain relief verified
* [ ] Cable tension minimized

---

# 2. Optical Readout Verification

* [ ] Laser securely mounted
* [ ] Laser spot centered on measurement surface
* [ ] Wall/screen distance measured and recorded
* [ ] Laser drift acceptable during 5-minute baseline
* [ ] Optical path unobstructed
* [ ] Spot visibility adequate for measurement
* [ ] Calibration geometry recorded

---

# 3. Electrical System Verification

* [ ] Power supply OFF before setup
* [ ] Voltage range verified
* [ ] Current limit configured
* [ ] Wiring polarity verified
* [ ] All exposed conductors secured
* [ ] Grounding configuration documented
* [ ] Dummy load available for controls
* [ ] Connector integrity checked

---

# 4. Environmental Controls

* [ ] Ambient temperature recorded
* [ ] Humidity recorded
* [ ] Airflow minimized
* [ ] Fans/HVAC disturbances minimized
* [ ] Table vibration acceptable
* [ ] Nearby magnetic sources removed
* [ ] Electrostatic buildup minimized
* [ ] External disturbances documented

---

# 5. Calibration Procedures

* [ ] Baseline drift run completed
* [ ] Static-response sensitivity verified
* [ ] Laser displacement scaling verified
* [ ] Noise floor characterized
* [ ] Calibration constants recorded
* [ ] Recovery behavior acceptable

---

# 6. Control-Test Readiness

* [ ] Dummy-load control prepared
* [ ] Polarity-reversal sequence prepared
* [ ] Geometry-reversal sequence prepared
* [ ] Power-off baseline sequence prepared
* [ ] Environmental monitoring active

---

# 7. Data Logging Preparation

* [ ] Run identifier assigned
* [ ] Timestamp synchronization verified
* [ ] CSV logging template prepared
* [ ] Notes section initialized
* [ ] Test operator recorded
* [ ] Experimental configuration documented

---

# 8. Safety Verification

* [ ] Power supply within safe operating range
* [ ] No exposed hazardous wiring
* [ ] Thermal risks minimized
* [ ] Emergency power disconnect accessible
* [ ] Workspace clear of conductive debris

---

# 9. Final Go / No-Go Assessment

* [ ] Baseline stable within acceptable drift limits
* [ ] No uncontrolled oscillation observed
* [ ] Calibration valid
* [ ] Environmental conditions acceptable
* [ ] Controls ready
* [ ] Data logging active

---

# Run Status

* [ ] APPROVED FOR TESTING
* [ ] HOLD / INVESTIGATE BEFORE RUN

# Philosophy

This checklist exists to support reproducibility-oriented precision residual-force metrology under conservative experimental methodology. Null results are scientifically valuable and help constrain candidate EVRT-like response amplitudes under calibrated diagnostic conditions.
