# Experimental Packages

This directory contains reproducibility-oriented EVRT experimental metrology packages.

Current focus areas include:

- precision residual-force null testing
- uncertainty-aware diagnostics
- calibration-oriented methodology
- artifact discrimination
- reproducibility workflows

## Experimental Philosophy

These packages are designed as conservative null-test-first frameworks emphasizing:

- calibration discipline
- uncertainty reduction
- reproducibility
- systematic artifact rejection
- conservative interpretation of residual signatures

Null results remain scientifically meaningful because they constrain candidate EVRT-like response amplitudes and improve future metrology workflows.


---

## Current Limitations

Current limitations include:
- sensitivity constraints
- environmental coupling
- limited voltage regime
- absence of vacuum operation
- lack of independent replication
