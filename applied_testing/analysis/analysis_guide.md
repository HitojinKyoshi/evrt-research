# Analysis Guide

## Objective

To determine whether any observed motion or force is:

- repeatable
- geometry-dependent
- independent of known artifacts

## Basic Plots

1. Position vs Time
2. Power ON vs OFF comparison
3. Orientation comparison (0° vs 180°)
4. Temperature vs position

## Key Questions

- Does motion begin when power is applied?
- Does motion stop when power is removed?
- Does direction reverse when orientation is flipped?
- Does motion appear in dummy or symmetric setups?

## Red Flags (Artifacts)

- motion correlates with temperature only
- motion changes with cable position
- motion present in dummy load
- motion same direction regardless of orientation

## Candidate Signal Indicators

- repeatable across runs
- reverses with geometry
- absent in controls
- scales with voltage or configuration

## Output

Summarize results as:

- Null
- Artifact-dominated
- Candidate signal (requires further validation)
