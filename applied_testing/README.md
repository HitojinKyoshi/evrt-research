# Applied EVRT Testing Program

This section extends the EVRT framework into applied experimental testing.

The objective is to determine whether asymmetric nonequilibrium electromagnetic systems can produce repeatable residual-force signatures after systematic elimination of conventional artifacts.

## Scope

This program focuses on:

- residual-force mapping
- orientation-reversal testing
- dummy-load and symmetric controls
- thermal, cable, vibration, and electromagnetic artifact rejection
- scaling behavior and parameter sensitivity
- momentum-transfer considerations

## Scientific Position

This work does NOT assume the existence of anomalous forces.

Any observed signal is treated as artifact-dominated unless it:

- persists under controlled conditions
- reverses with geometry
- does not appear in control configurations
- remains reproducible across trials

## Philosophy

The goal is not to prove propulsion, but to establish whether any measurable effect survives strict falsification.

Null results are scientifically meaningful and contribute to constraint-building.
