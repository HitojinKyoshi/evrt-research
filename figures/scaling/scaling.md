# Scaling Figures

This folder contains scaling-law visualizations, parameter-space maps, and normalization graphics associated with the EVRT framework.

These figures support:

- residual-force scaling analysis
- parameter-bound interpretation
- Q-factor sensitivity evaluation
- effective susceptibility normalization
- uncertainty-aware metrology planning
- simulation-guided parameter exploration

---

# Included Figure Types

Typical figures include:

- force-versus-Q scaling plots
- signal-to-noise simulations
- effective susceptibility maps
- parameter-space bounds
- power-scaling trends
- energy-density normalization graphics

# Scientific Position

The scaling figures contained in this directory are phenomenological and exploratory unless explicitly tied to experimentally calibrated measurements.

These figures are intended to support:

- falsifiable hypothesis testing
- sensitivity estimation
- parameter-bound analysis
- scaling-law comparison
- residual-force normalization workflows

The EVRT framework does not claim experimentally verified anomalous-force phenomena.
