# Experimental Protocol

## Objective

Measure whether asymmetric driven electromagnetic architectures produce reproducible torsional responses above the calibrated noise floor after environmental and instrumentation artifacts are constrained.

## Apparatus Baseline

- Suspension: nonconductive fishing line
- Arm length: ~1 ft
- Laser projection distance: ~3 ft
- Electrode gaps tested: 2 cm and 5 cm
- Power range: 0–31 V
- Baseline observation: no measurable torsional response up to 31 V

## Experimental Sequence

1. Mechanical baseline stabilization
2. Laser alignment and drift characterization
3. Power-off baseline run
4. Dummy-load control test
5. Polarity reversal test
6. Geometry reversal test
7. Powered experimental run
8. Thermal cooldown observation
9. Repeatability verification

## Required Logging

- ambient temperature
- humidity
- applied voltage
- current draw
- torsional displacement
- timestamp
- environmental disturbances

## Success Criteria

A candidate signal must survive:
- polarity reversal
- geometry reversal
- dummy-load substitution
- thermal stabilization
- repeatability testing
- uncertainty analysis
