# Transmitter Build Notes

## Configuration

- Large plate: 3" x 3" aluminum
- Small plate: 1" x 1" aluminum
- Gap: 3–5 mm (plastic spacers)
- Resistor: 10Ω or 20Ω (50W)
- Voltage: 12–24V

## Layout

- Plates mounted rigidly on nonconductive base
- Resistor placed away from plates
- Wires strain-relieved before connection
- Slack loop between device and power supply

## Key Constraints

- No cable tension on platform
- Stable mounting
- Controlled gap spacing
- Power supply off platform

## Notes

This build is intended for controlled, low-voltage testing only.
High-voltage operation is not part of this configuration.
